# 🦁 Симуляция саванны

Африканская саванна оживает — пошаговая 2D‑симуляция, где травоядные ищут траву, хищники охотятся, а окружающая среда влияет на выживание.  
Проект написан на Python с использованием **PySide6** для графического интерфейса и демонстрирует принципы ООП, паттерны **Factory** и **Observer**, а также алгоритм поиска пути BFS.

---

## 🎮 Возможности

- 🗺 **Динамическая карта** – размер 20×20 клеток (настраивается в коде).
- 🦒 **Два типа существ** – травоядные (зебры, жирафы, антилопы) и хищники (львы, леопарды, гиены).
- 🌿 **Ресурсы и препятствия** – трава (пища), камни и деревья (непроходимые объекты).
- 🧠 **Искусственный интеллект** – каждое существо самостоятельно находит ближайшую цель с помощью BFS, учитывает скорость и атакует/поедает при достижении цели.
- ⏱ **Пошаговый режим и автоматическая симуляция** – можно запустить бесконечный цикл или делать ходы вручную.
- 📊 **Информационная панель** – отображает текущий ход, количество травоядных, хищников, травы.
- 🖱 **Интерактивная карта** – при наведении на клетку показывается подсказка с типом существа и его характеристиками (HP, скорость, сила атаки).
- 🔄 **Сброс симуляции** – возвращает мир в начальное состояние с заданным количеством существ.

## 🚀 Установка и запуск

### Требования
- Python 3.10 или выше
- Библиотека PySide6

### Установка зависимостей

```bash
pip install pyside6
```
### Запуск
```bash
python main.py
```
После запуска взаимодействие с приложением через графический интерфейс

### Архитектура проекта
Проект разделён на три основные пакета:

core/

- coordinates.py – неизменяемый класс координат.
- game_map.py – хранилище сущностей, проверка проходимости.
- renderer.py – отрисовка карты (QGraphicsView).
- simulation.py – главный управляющий класс (таймер, ходы, сигналы).

entities/
- base/ – абстрактные классы Entity и Creature.

- herbivores/ – травоядные (Herbivore, Antelope, Giraffe, Zebra).

- predators/ – хищники (Predator, Lion, Leopard, Hyena).

- static/ – статичные объекты (Grass, Rock, Tree).

- entity_factory.py – фабрика для создания сущностей со случайными параметрами

gui/
- main_window.py – главное окно, диалоги, обработка кнопок.

- main_window_template2.py – сгенерированный из .ui интерфейс.

- validation.py – проверка ввода.

- constants.py – значения по умолчанию.
